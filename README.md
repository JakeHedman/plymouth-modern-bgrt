
# plymouth-bgrt theme

A theme for plymouth which uses your system's UEFI logo (aka BGRT).

## Installation

```sh
    sudo ./install.sh	# Fetches your BGRT, adjusts the theme to suit and installs it.
    sudo plymouth-set-default-theme -R plymouth-bgrt
```
